# GHUD Lite - Garmin HUD для Android

Современное приложение для работы с Garmin HUD, поддерживающее навигацию из Яндекс.Навигатора и Google Maps, а также данные об ограничениях скорости и камерах из OpenStreetMap.

## 🎯 Основные возможности

### Навигация
- ✅ **Поддержка Яндекс.Навигатор** - парсинг данных через Accessibility Service
- ✅ **Поддержка Google Maps** - парсинг данных через Notification Listener
- ✅ **Распознавание направлений** - 9 типов стрелок (прямо, налево, направо, плавно, резко, держаться)
- ✅ **Правильное форматирование расстояний** - автоматическое переключение м/км
- ✅ **Время прибытия (ETA)** - отображение на HUD
- ✅ **Оставшееся время** - до конца маршрута

### Безопасность
- ✅ **OpenStreetMap интеграция** - ограничения скорости с дорог
- ✅ **Камеры контроля скорости** - предупреждение за 300м
- ✅ **Индикатор превышения скорости** - настраиваемый порог
- ✅ **Текущая скорость** - с GPS в реальном времени

### HUD
- ✅ **Автоматическое подключение** - к последнему устройству
- ✅ **Автояркость** - или ручная настройка
- ✅ **Настраиваемые профили** - разные для навигации и обычной езды
- ✅ **Отображение времени** - на HUD дисплее

### Отладка
- ✅ **Debug Activity** - полная диагностика работы приложения
- ✅ **Виртуальный HUD** - предпросмотр отображаемой информации
- ✅ **Внутренние логи** - с цветовой индикацией уровня важности
- ✅ **Мониторинг всех сервисов** - OSM, Navigation, HUD

## 🚀 Быстрый старт

### Сборка и установка

```powershell
# Один скрипт для всего (компиляция + установка)
.\build_and_install.ps1
```

Или вручную:
```powershell
# Сборка
.\gradlew assembleDebug

# Установка
adb install -r app\build\outputs\apk\debug\app-debug.apk
```

### Первоначальная настройка

1. **Предоставьте разрешения:**
   - GPS (для скорости и координат)
   - Bluetooth (для подключения к HUD)
   - Accessibility Service (для Яндекс.Навигатора)
   - Notification Listener (для Google Maps)

2. **Подключите Garmin HUD:**
   - Включите HUD
   - Нажмите "Подключить HUD" в приложении
   - Выберите устройство из списка

3. **Начните использование:**
   - Запустите навигацию в Яндекс.Навигаторе или Google Maps
   - Данные автоматически появятся на HUD

## 🔧 Архитектура

### Распознавание стрелок (двухуровневое)

**Уровень 1: По изображению** (приоритет)
- AccessibilityService → скриншот ImageView
- ArrowImage → вычисление hash (8x8 grid)
- ArrowDirection.recognize() → сравнение с эталонами

**Уровень 2: По тексту** (fallback)
- Парсинг инструкции ("поверните налево")
- Поддержка 9 типов + русский/английский

### Универсальное состояние

```kotlin
object HudState {
    var isNavigating: Boolean           // Активна ли навигация
    var distanceToTurn: String?         // "500 m", "1.5 км"
    var distanceToTurnMeters: Int?      // 500, 1500
    var turnIcon: Int?                  // 0-8 (код направления)
    var currentSpeed: Int               // км/ч
    var speedLimit: Int?                // км/ч
    var isSpeeding: Boolean             // Превышение
    var cameraDistance: Int?            // Расстояние до камеры (м)
}
```

## 🐛 Решение проблем

### HUD не подключается
- Проверьте включён ли Bluetooth
- Убедитесь что HUD в режиме сопряжения
- Предоставьте все Bluetooth разрешения

### Не работает навигация из Яндекса
- Включите Accessibility Service в настройках Android
- Перезапустите приложение

### Не отображается ограничение скорости
- Проверьте GPS сигнал
- Убедитесь что есть интернет
- Не все дороги размечены в OSM

## 📄 Лицензия

Проект является модификацией GoogleMaps_HUD от sky4s с расширенным функционалом.

## 🤝 Вклад

Для сбора эталонных hash-значений стрелок Яндекса:
1. Запустите навигацию
2. Откройте Debug Activity
3. В Internal Logs найдите значения Hash
4. Сопоставьте с типом поворота
5. Отправьте данные разработчику
